### Download

```bash
git clone https://github.com/DimaMSISur/templateFastAPI.git && cd templateFastAPI
```

### Usage

```bash
poetry install
```

```bash
poetry run python src/main.py
```

**_If you have not installed poetry, install it_**

```bash
sudo apt install poetry
```

**_or_**

```bash
sudo dnf install poetry
```
